# haiou
wget -O /usr/local/bin/haiou https://raw.githubusercontent.com/Eranther/haiou/main/haiou.sh && chmod +x /usr/local/bin/haiou && haiou

## 功能

- 安装 / 重装 VLESS + REALITY + XHTTP
- 生成 Clash Verge / Mihomo YAML 订阅
- 不重装节点，重新生成 Clash Verge YAML
- 查看节点摘要 / 完整节点信息 / 二维码
- 删除节点配置
- 更新 Xray / 更新脚本 / 卸载
